Lệnh 1 (Tạo khóa bí mật - Private Key):

Bash
openssl genpkey -algorithm RSA -out private.pem -pkeyopt rsa_keygen_bits:2048


Lệnh 2 (Trích xuất khóa công khai - Public Key):

Bash
openssl rsa -pubout -in private.pem -out public.pem
